There are 5 to 6 types of element sizings

There are mainly 2 categories of sizing types --->
1. Absolute
2. Relative

1. px ---> it helps in sizing the elements with a absolute size

2. % ---> it helps in sizing the elements with respect to the parent elements

3. rem --> it helps in sizing the elements with respect to the parent

3. em --> it helps in sizing the elements with respect to the grandchild

4. vw ---> it helps in sizing the elements with respect to the viewport width

4. vh ---> it helps in sizing the elements with respect to the viewport height
